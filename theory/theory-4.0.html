PHASE 4.0 THEORY

Transient Nonlinear Structural Dynamics

Direct integration of the full-order model (FOM) using Newmark-β.

The Governing Equation

Phase 4.0 extends our nonlinear static solver into the time domain by including inertial and dissipative effects:

$$\mathbf{M} \ddot{\mathbf{u}} + \mathbf{C} \dot{\mathbf{u}} + \mathbf{F}_{int}(\mathbf{u}) = \mathbf{F}_{ext}(t)$$

This is a second-order nonlinear ODE. To achieve real-time performance in the Digital Twin, we must first solve this expensive FOM to collect snapshots.

1. Mass and Damping Matrices

The global mass matrix $\mathbf{M}$ is assembled using consistent IGA shape functions, ensuring accurate kinetic energy representation. We employ Rayleigh Damping for numerical stability:

$$\mathbf{C} = \alpha \mathbf{M} + \beta_R \mathbf{K}_t$$

This proportional damping allows us to control low-frequency and high-frequency oscillations independently during the transient response.

2. The Newmark-β Algorithm

We use an implicit, unconditionally stable scheme to integrate the equations of motion. At each step $n+1$, we solve for displacement using an iterative Newton-Raphson loop:

  • Predictor: Estimate $\mathbf{u}_{n+1}$ using velocity and acceleration from the previous step.
  • Equilibrium: Calculate the residual $\mathbf{R} = \mathbf{M}\ddot{\mathbf{u}} + \mathbf{C}\dot{\mathbf{u}} + \mathbf{F}_{int} - \mathbf{F}_{ext}$.
  • Effective Stiffness: $\mathbf{K}_{eff} = \mathbf{K}_t + a_0 \mathbf{M} + a_1 \mathbf{C}$.
  • Correction: $\Delta \mathbf{u} = -\mathbf{K}_{eff}^{-1} \mathbf{R}$.

Numerical Stability

By setting $\beta = 0.25$ and $\gamma = 0.5$, the scheme becomes the Average Acceleration Method, which is energy-conserving and prevents non-physical damping of lower modes.
