1. The Offline-Online Decomposition

To achieve real-time performance, the computational burden must be split into two distinct phases. This methodology is the cornerstone of Model Order Reduction (MOR).

  • Offline Stage (Heavy): High-fidelity FOM simulations are run to build a Snapshot Matrix ($S$). Proper Orthogonal Decomposition (POD) via SVD extracts the orthonormal basis $\Phi$. Operators are then projected onto this basis.
  • Online Stage (Light): Only the reduced equations are solved. The system size drops from $N_{dof} \approx 5,000$ to $k \approx 10$, enabling 60+ FPS interaction.

2. The Affine Parametric Expansion

A static ROM only works for a fixed material. To enable sliders for $E$ and $\nu$ in the UI without re-projecting the whole system, we use an Affine Decomposition of the stiffness matrix:

$$K(E, \nu) = \frac{E}{1-\nu^2} \left[ K_1 + \nu K_2 \right]$$

By pre-calculating the projected matrices $K_{r,1} = \Phi^T K_1 \Phi$ and $K_{r,2} = \Phi^T K_2 \Phi$ during the offline phase, the online solver only needs to perform a simple scalar addition and solve a tiny $k \times k$ system:

$$\left( \frac{E}{1-\nu^2} (K_{r,1} + \nu K_{r,2}) \right) \mathbf{q} = \mathbf{F}_r$$

3. High-Fidelity Reconstruction

Once the reduced coordinates $\mathbf{q}$ are found, the full physical field (displacements or stresses) is reconstructed using the projection matrix:

$$\mathbf{u} \approx \Phi \mathbf{q}$$

For the Digital Twin visualization, we then perform a Numerical Stress Recovery by evaluating the B-matrix at each visual vertex, allowing us to see Von Mises profiles in real-time as we interact with the physical parameters.
