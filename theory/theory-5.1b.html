Phase 5.1b: ECSW Hyper-reduction

Detailed Mathematical Formulation and Computational Algorithms for Real-Time Shape-Parameterized ROMs

Problem Statement: Standard POD-Galerkin projection requires a full mesh assembly step ($O(N_e)$) for every new parametric geometric shape $\mathbf{\theta}$. This completely negates the real-time speedup during online evaluation. Energy-Conserving Sampling and Weighting (ECSW) resolves this assembly bottleneck by performing a sparse integration ($O(|E_{\text{sample}}|)$) over a mathematically optimized subset of elements, providing up to $100\times$ speedups while guaranteeing physical stability and energy conservation.

1. Exact NURBS Bivariate Geometry Morphing

In Isogeometric Analysis (IGA), the physical coordinate domain $\Omega(\mathbf{\theta})$ is parametrized by a shape variable $\mathbf{\theta}$. The mapping from the parametric space $\mathbf{\xi} = (\xi, \eta)$ to the physical coordinates $\mathbf{x} = (x, y)$ is governed by the same rational basis functions used to represent the geometry (NURBS):

$$\mathbf{x}(\mathbf{\xi}; \mathbf{\theta}) = \sum_{i=1}^n \sum_{j=1}^m R_{i,j}^{p,q}(\mathbf{\xi}) \mathbf{P}_{i,j}(\mathbf{\theta})$$

where $\mathbf{P}_{i,j}(\mathbf{\theta})$ are the morphed NURBS control points, and $R_{i,j}^{p,q}(\mathbf{\xi})$ are the bivariate rational shape functions defined as:

$$R_{i,j}^{p,q}(\xi, \eta) = \frac{N_{i,p}(\xi) M_{j,q}(\eta) \omega_{i,j}}{\sum_{r=1}^{n} \sum_{s=1}^{m} N_{r,p}(\xi) M_{s,q}(\eta) \omega_{r,s}}$$

The **Jacobian Matrix** $\mathbf{J}(\mathbf{\xi}; \mathbf{\theta})$ transforms derivatives from the parametric space to the physical space:

$$\mathbf{J}(\mathbf{\xi}; \mathbf{\theta}) = \frac{\partial \mathbf{x}}{\partial \mathbf{\xi}} = \begin{bmatrix} \frac{\partial x}{\partial \xi} & \frac{\partial x}{\partial \eta} \\ \frac{\partial y}{\partial \xi} & \frac{\partial y}{\partial \eta} \end{bmatrix}$$

The gradient of the NURBS shape functions in physical coordinates is computed by transforming their parametric gradients with the inverse Jacobian $\mathbf{J}^{-1}(\mathbf{\xi}; \mathbf{\theta})$:

$$\nabla_{\mathbf{x}} R_{A} = \mathbf{J}^{-1}(\mathbf{\xi}; \mathbf{\theta}) \nabla_{\mathbf{\xi}} R_{A}$$

2. Bivariate Stiffness Assembly and the Bottleneck

For a linear elastic material, the global stiffness matrix $\mathbf{K}(\mathbf{\theta})$ is assembled as a sum of local element stiffness matrices $\mathbf{K}_e(\mathbf{\theta})$:

$$\mathbf{K}(\mathbf{\theta}) = \sum_{e=1}^{N_e} \mathbf{K}_e(\mathbf{\theta})$$

Each element contribution is integrated numerically using Gauss-Legendre quadrature over the bivariate knot span $\widehat{\Omega}_e = [\xi_i, \xi_{i+1}] \times [\eta_j, \eta_{j+1}]$:

$$\mathbf{K}_e(\mathbf{\theta}) = \sum_{a=1}^{n_{GP}} \sum_{b=1}^{m_{GP}} \mathbf{B}_e^T(\tilde{\xi}_a, \tilde{\eta}_b; \mathbf{\theta}) \mathbf{C} \mathbf{B}_e(\tilde{\xi}_a, \tilde{\eta}_b; \mathbf{\theta}) \left| \det \mathbf{J}(\tilde{\xi}_a, \tilde{\eta}_b; \mathbf{\theta}) \right| w_a w_b$$

where $\mathbf{C}$ is the elastic material constitutive tensor, and $\mathbf{B}_e$ is the strain-displacement operator.

In standard Galerkin projection, we define a low-dimensional displacement subspace spanned by the Proper Orthogonal Decomposition (POD) basis $\mathbf{\Phi} \in \mathbb{R}^{N_{dof} \times k}$. The reduced equations are:

$$\mathbf{k}_r(\mathbf{\theta}) \mathbf{a} = \mathbf{\Phi}^T \mathbf{f}$$

where:

$$\mathbf{k}_r(\mathbf{\theta}) = \mathbf{\Phi}^T \mathbf{K}(\mathbf{\theta}) \mathbf{\Phi} = \sum_{e=1}^{N_e} \mathbf{\Phi}_e^T \mathbf{K}_e(\mathbf{\theta}) \mathbf{\Phi}_e$$

The Assembly Bottleneck: Because the shape parameter $\mathbf{\theta}$ is continuously morphed by the user, the element Jacobian matrices $\mathbf{J}_e$ and strain-displacement operators $\mathbf{B}_e$ are completely variable. Consequently, $\mathbf{k}_r(\mathbf{\theta})$ must be recomputed by looping over all $N_e$ elements, projecting each element stiffness matrix $\mathbf{K}_e(\mathbf{\theta})$ onto the basis $\mathbf{\Phi}_e$. This full mesh dependency scales as $O(N_e)$, destroying the speed advantages of the Reduced Order Model (ROM).

3. ECSW Core Mathematical Principles

The physical foundation of ECSW is the approximation of the virtual work of internal forces. For any virtual displacement state $\mathbf{u}^* = \mathbf{\Phi} \mathbf{a}^*$, the virtual work is preserved by choosing a sparse subset of elements $E_{\text{sample}} \subset \{1, \dots, N_e\}$ and positive scaling weights $w_e > 0$:

$$\mathbf{k}_r(\mathbf{\theta}) \approx \mathbf{k}_{r, \text{ECSW}}(\mathbf{\theta}) = \sum_{e \in E_{\text{sample}}} w_e \mathbf{\Phi}_e^T \mathbf{K}_e(\mathbf{\theta}) \mathbf{\Phi}_e$$

The Virtual Force System ($Aw \approx b$)

We require that the action of the approximated stiffness matrix on the basis vectors matches the full-order matrix action over a set of training parameter snapshots $\mathbf{\theta}_i$ ($i = 1, \dots, N_s$). For each snapshot $i$ and basis column $\mathbf{\phi}_j$ ($j = 1, \dots, k$), the virtual force contributions are:

$$\mathbf{A}_{e, i, j} = \mathbf{\Phi}^T \mathbf{K}_e(\mathbf{\theta}_i) \mathbf{\phi}_j \in \mathbb{R}^k$$ $$\mathbf{b}_{i, j} = \mathbf{\Phi}^T \mathbf{K}(\mathbf{\theta}_i) \mathbf{\phi}_j \in \mathbb{R}^k$$

By vectorizing and stacking these vectors for all basis columns and training snapshots, we define a global virtual force matrix $\mathbf{A} \in \mathbb{R}^{D \times N_e}$ and a target vector $\mathbf{b} \in \mathbb{R}^D$, where $D = N_s \cdot k^2$:

$$\mathbf{A} = \begin{bmatrix} \mathbf{A}_1 & \mathbf{A}_2 & \dots & \mathbf{A}_{N_e} \end{bmatrix}, \quad \mathbf{b} = \sum_{e=1}^{N_e} \mathbf{A}_e$$

The optimal element weights $\mathbf{w}$ are found by solving the following **Non-Negative Least Squares (NNLS)** problem:

$$\min_{\mathbf{w} \ge \mathbf{0}} \left\| \sum_{e=1}^{N_e} w_e \mathbf{A}_e - \mathbf{b} \right\|_2^2$$

Key Properties of ECSW Formulation:

  • Guaranteed Stability: The constraint $\mathbf{w} \ge \mathbf{0}$ ensures that the hyper-reduced stiffness matrix $\mathbf{k}_{r, \text{ECSW}}$ remains positive semi-definite (and positive definite under standard boundary conditions), ensuring structural physical stability.
  • Sparsity and Carathéodory's Theorem: The target vector $\mathbf{b}$ lies in the conic hull of the columns of $\mathbf{A}$. According to Carathéodory's theorem, any vector in the cone can be represented as a conic combination of at most $D + 1$ linearly independent columns. Hence, a sparse set of active elements $|E_{\text{sample}}| \le N_s \cdot k^2 + 1$ is mathematically guaranteed.

4. Computational Algorithms

The implementation is split into a robust offline training sweep and a real-time online projection solver.

Algorithm 1: Offline ECSW Hyper-reduction Training
1. Input: Snapshot parameter set $\Theta = \{\mathbf{\theta}_1, \dots, \mathbf{\theta}_{N_s}\}$, POD Basis $\mathbf{\Phi}$, tolerance $\text{tol}$.
2. Construct System:
   For each snapshot $\mathbf{\theta}_i \in \Theta$:
      a. Morph NURBS geometry and compute all full element stiffness matrices $\mathbf{K}_e(\mathbf{\theta}_i)$.
      b. For each basis column $j \in \{1,\dots, k\}$, assemble element virtual forces:
         $\mathbf{A}_{e, i, j} = \mathbf{\Phi}^T \mathbf{K}_e(\mathbf{\theta}_i) \mathbf{\phi}_j$
   c. Stack virtual forces to build global matrix $\mathbf{A} \in \mathbb{R}^{D \times N_e}$ and target vector $\mathbf{b} = \sum_{e=1}^{N_e} \mathbf{A}_e$.
3. Greedy Active-Set NNLS:
   a. Initialize active index set $\mathcal{I} = \emptyset$, residual vector $\mathbf{r} = \mathbf{b}$, weights $\mathbf{w} = \mathbf{0}$.
   b. For $\text{iter} = 1$ to $\min(N_e, 200)$:
         i. Identify best matching element: $e^* = \arg\max_{e \notin \mathcal{I}} \left( \mathbf{A}_e^T \mathbf{r} \right)$.
         ii. If $\mathbf{A}_{e^*}^T \mathbf{r} \le 0$, break loop.
         iii. Update active set: $\mathcal{I} = \mathcal{I} \cup \{e^*\}$.
         iv. Solve unconstrained least squares: $\mathbf{w}_{\mathcal{I}} = (\mathbf{A}_{\mathcal{I}}^T \mathbf{A}_{\mathcal{I}})^{-1} \mathbf{A}_{\mathcal{I}}^T \mathbf{b}$.
         v. Enforce non-negativity: $w_e = \max(0, w_e)$ for all $e \in \mathcal{I}$.
         vi. Recalculate residual: $\mathbf{r} = \mathbf{b} - \sum_{e \in \mathcal{I}} w_e \mathbf{A}_e$.
         vii. Check relative convergence: if $\|\mathbf{r}\|_2 / \|\mathbf{b}\|_2 < \text{tol}$, break.
4. Output: Sampled active elements $E_{\text{sample}} = \{e \in \mathcal{I} \mid w_e > 0\}$ and their weights $\mathbf{w}_{E_{\text{sample}}}$.
Algorithm 2: Online ECSW Hyper-reduced Solve
1. Input: Morphing parameter $\mathbf{\theta}_{\text{new}}$, Active elements $E_{\text{sample}}$, weights $\mathbf{w}$, Basis $\mathbf{\Phi}$.
2. Sparse Assembly:
   Initialize $\mathbf{k}_r = \mathbf{0} \in \mathbb{R}^{k \times k}$.
   For each element $e \in E_{\text{sample}}$:
      a. Morph local element control points: $\mathbf{P}_{e}(\mathbf{\theta}_{\text{new}})$.
      b. Evaluate Jacobian $\mathbf{J}_e$ and strain operator $\mathbf{B}_e$ at quadrature points.
      c. Assemble local element stiffness: $\mathbf{K}_e(\mathbf{\theta}_{\text{new}})$.
      d. Project and accumulate: $\mathbf{k}_r += w_e \mathbf{\Phi}_e^T \mathbf{K}_e(\mathbf{\theta}_{\text{new}}) \mathbf{\Phi}_e$.
3. Project Load Vector: Compute $\mathbf{f}_r = \mathbf{\Phi}^T \mathbf{f}$.
4. Reduced Solve: Solve the $k \times k$ linear system $\mathbf{k}_r \mathbf{a} = \mathbf{f}_r$.
5. Reconstruct: Reconstruct full displacement field: $\mathbf{u}_{ROM} = \mathbf{\Phi} \mathbf{a}$.
6. Output: Real-time stresses and deformations.
