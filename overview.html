RESEARCH ABSTRACT

Project Overview

Digital Twin methodology for FSI / Non-Linear dynamics courses.

The Laboratory of Structural Mechanics and Coupled Systems (LMSSC) at Cnam is dedicated to studying the coupled dynamic behavior between multi-physical systems. This project bridges structural mechanics and interactive software by focusing on reduced order modelling of nonlinear dynamic problems with geometrical parameters.

Core Objectives

The main goal is to propose a methodology and validate a **Reduced Order Model (ROM)** strategy to solve nonlinear dynamic problems with large displacements of structures. We use **Isogeometric Analysis (IGA)** and innovative hyper-reduction techniques to construct the ROM, ensuring both accuracy and computational efficiency for real-time applications.

Real-time Synergy

The originality of this core research consists in its pedagogical application. We aim to create a "Digital Twin" set-up that integrates a physical 3D-printed model with a real-time virtual simulation environment. This hybrid approach allows students and researchers to observe the correlation between theoretical models and physical behavior in dynamic environments like Fluid-Structure Interaction (FSI).
