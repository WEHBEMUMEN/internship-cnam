Reduced Order Modeling (ROM)

The core justification for Isogeometric Analysis in the Digital Twin era.

What is ROM?

In engineering, Reduced Order Modeling (ROM) is the process of simplifying a complex, high-fidelity mathematical model into a lower-dimensional representation that retains the essential physics.

A typical Finite Element Model (FEM) of a bridge or wing might have millions of degrees of freedom (DoFs). Solving this in real-time for a Digital Twin is impossible. ROM allows us to "compress" these millions of equations into a few hundred, enabling real-time feedback.

Why IGA is the Perfect ROM

Traditional FEM uses piecewise linear triangles or boxes. To represent a smooth curve accurately, you need thousands of these small elements.

Isogeometric Analysis (IGA) uses NURBS—the same smooth curves used in CAD. Because NURBS are inherently smooth (C² continuous), a single large "element" can represent what FEM needs 100 elements for.

Classical FEM

  • Linear approximations
  • High parameter count
  • Discontinuous gradients
  • Slow for Digital Twins

Isogeometric ROM

  • Exact Geometry
  • Minimal Parameters (DoFs)
  • Smooth gradients (C²)
  • Real-time performance

The Result

In our simulator, you can see the **Reference Solution** (white dashed line) being matched almost perfectly by the **ROM Curve** (blue line).

While the Reference mesh requires 202 DoFs, our IGA model achieves the same shape with only 8 DoFs. That is a 96% reduction in computational complexity without losing structural fidelity.
