Euler-Bernoulli Theory & Derivation

The beam behavior is governed by local equations relating moment $M$, shear $V$, and load $q$. In the Phase 1.7 engine, we derive the Weak Form using the principle of virtual work.

Equilibrium: $M_{,xx} + q = 0$
Constitutive: $M = -EI w_{,xx}$

To derive the weak form, we multiply by a virtual displacement $\delta w$ and integrate. Through Integration by Parts (IPP), derivatives are transferred:

$$\int_0^L (M_{,xx} + q) \delta w \, dx = 0$$ $$\text{IPP1: } -\int_0^L M_{,x} \delta w_{,x} \, dx + [M_{,x} \delta w]_0^L + \int_0^L q \delta w \, dx = 0$$ $$\text{IPP2: } \int_0^L M \delta w_{,xx} \, dx - [M \delta w_{,x}]_0^L + \int_0^L q \delta w \, dx = 0$$

With appropriate boundary conditions, the final weak form for flexural bending is: $$\int_0^L EI w_{,xx} \delta w_{,xx} \, dx = \int_0^L q(x) \delta w \, dx$$

Mathematics: Knot Vectors & Basis Functions

In the Phase 1.7 engine, the geometry is defined using Non-Uniform Rational B-Splines (NURBS). The domain is dictated by a Knot Vector ($\Xi$), which divides the parametric space into elements.

Example Knot Vector: $\Xi = \{0, 0, 0, \xi_a, \xi_b, \xi_c, 1, 1, 1\}$

The engine uses an Open Uniform Knot Vector. This means the ends are clamped with $p+1$ repeating values, forcing interpolation at the boundaries. The basis functions are computed via the Cox-de Boor formula:

$$N_{i,0}(\xi) = \begin{cases} 1 & \text{if } u_i \le \xi < u_{i+1} \\ 0 & \text{otherwise} \end{cases}$$ $$N_{i,p}(\xi)=\frac{\xi - u_i}{u_{i+p} - u_i} N_{i,p-1}(\xi) + \frac{u_{i+p+1} - \xi}{u_{i+p+1} - u_{i+1}} N_{i+1,p-1}(\xi)$$

For precise geometric circularity, we use weights $w_i$: $R_{i,p}(\xi) = \frac{N_{i,p}(\xi) w_i}{\sum N_{j,p}(\xi) w_j}$.

Isoparametric Transformation & Jacobian

We map the physical domain $x$ to a parametric domain $\xi \in [0, 1]$. Because the Euler-Bernoulli weak form requires second derivatives, the Jacobian ($J = x^h_{,\xi}$) plays a critical role.

Chain Rule: $w_{,x} = \frac{1}{J} w^h_{,\xi} \quad ; \quad w_{,xx} = \frac{1}{J^2} w^h_{,\xi\xi}$

The resulting stiffness matrix $\mathbf{K}$ requires integration with the Jacobian to the third power:

$$K_{ij} = \int_0^1 EI \left( \frac{1}{J^2} B_i \right) \left( \frac{1}{J^2} B_j \right) J \, d\xi = \int_0^1 EI \frac{B_i B_j}{J^3} \, d\xi$$

The C¹ Necessity

If degree $p=1$ (Linear), then $w^h_{,\xi\xi} = 0$. Flexural bending requires $p \ge 2$ to maintain non-zero curvature within the elements.

Snapshot Matrix & SVD

We collect solutions for various parameters $\mu$ into a snapshot matrix $\mathbb{W} = [\underline{W}_1, \dots, \underline{W}_p]$. Singular Value Decomposition (SVD) identifies the dominant modes:

$$\text{SVD}(\mathbb{W}) \to U \Sigma V^T$$

Non-linear Manifolds: Autoencoders

For non-linear behaviors, we replace linear modal combinations with an Encoder-Decoder structure.

$\underline{W}(\mu) \approx \Gamma(\underline{q})$
$\underline{q}$: Latent coefficients (compressed state).

Force Positioning via Dirac Delta

IGA allows for precise loading at any continuous parametric coordinate $\xi \in [0, 1]$. Unlike traditional FEM, we don't need a node at the load point.

$$F_i = \int_{0}^{1} P \delta(\xi - \mu) R_i(\xi) J(\xi) \, d\xi = P R_i(\mu) J(\mu)$$

By sampling the basis functions at the exact point of application $\mu$, the point load is distributed smoothly across surrounding control points.

Structural Dynamics & Time Integration

In addition to static equilibrium ($\mathbf{K}u=\mathbf{F}$), the Phase 1.7 engine supports dynamic structural responses. By introducing the Mass Matrix ($\mathbf{M}$) and Damping Matrix ($\mathbf{C}$), we model the structure's behavior under time-varying forces and inertial effects.

Equation of Motion: $\mathbf{M} \mathbf{\ddot{u}}(t) + \mathbf{C} \mathbf{\dot{u}}(t) + \mathbf{K} \mathbf{u}(t) = \mathbf{F}(t)$

The mass matrix $\mathbf{M}$ integrates the density ($\rho$) and cross-sectional area ($A$) using the identical Isogeometric mapping workflow. This preserves the "exact geometry" parameterization directly into the inertia computations:

$$M_{ij} = \int_{0}^{1} \rho A \, R_i(\xi) R_j(\xi) J(\xi) \, d\xi$$

For the damping matrix $\mathbf{C}$, the engine implements Rayleigh Damping, assuming the damping is proportional to a linear combination of the mass and stiffness matrices. This drastically simplifies the evaluation of energy dissipation within the structure:

$$\mathbf{C} = \alpha \mathbf{M} + \beta \mathbf{K}$$ Where $\alpha$ (mass-proportional) targets low-frequency damping, and $\beta$ (stiffness-proportional) targets high-frequency damping.

Time Integration: The Newmark-$\beta$ Method

Evaluating the structural displacement over continuous time requires numerical time-stepping schemes. We employ the unconditionally stable, implicit Newmark-$\beta$ method, reformulating the dynamic equations into an equivalent static problem solved recursively at each time step $\Delta t$:

$$\hat{\mathbf{K}} \mathbf{u}_{t+\Delta t} = \hat{\mathbf{F}}_{t+\Delta t}$$ $$\hat{\mathbf{K}} = \mathbf{K} + \frac{1}{\beta \Delta t^2}\mathbf{M} + \frac{\gamma}{\beta \Delta t}\mathbf{C}$$

While highly stable, factorizing the dense $\hat{\mathbf{K}}$ iteratively within a web browser executing Isogeometric Analysis is extremely taxing on hardware.

Subspace Dynamics (The ROM Solution)

Because $\mathbf{M}$ and $\mathbf{K}$ possess wide bandwidths in IGA, direct implicit algorithms scale poorly for real-time visualization. Instead, we project the massive dynamic equation down into a tiny Modal Subspace via Galerkin Projection using our learned basis matrix $\mathbf{V}$:

$$\mathbf{M}_{red} \mathbf{\ddot{q}}(t) + \mathbf{C}_{red} \mathbf{\dot{q}}(t) + \mathbf{K}_{red} \mathbf{q}(t) = \mathbf{F}_{red}(t)$$

Here, the $N \times N$ matrices collapse to microscopic $m \times m$ dense matrices (where $m \ll N$). Time integration calculations now execute in microseconds, allowing instantaneous wave simulation and vibration tuning in a 60 FPS constraint.
