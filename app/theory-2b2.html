Newton-Raphson

Large deformations and the iterative path to equilibrium.

1. Geometric Nonlinearity

When a structure undergoes large rotations or displacements, the linear strain assumption ($\epsilon = \frac{1}{2}(\nabla u + \nabla u^T)$) is no longer valid. We must use the **Green-Lagrange Strain Tensor** $E$, which accounts for the quadratic terms of deformation:

$$E = \frac{1}{2}(\mathbf{F}^T \mathbf{F} - \mathbf{I})$$ $$E_{ij} = \frac{1}{2} \left( \frac{\partial u_i}{\partial X_j} + \frac{\partial u_j}{\partial X_i} + \frac{\partial u_k}{\partial X_i} \frac{\partial u_k}{\partial X_j} \right)$$

where $\mathbf{F} = \mathbf{I} + \nabla \mathbf{u}$ is the deformation gradient. This allows us to track the stiffness change caused by the current shape of the structure.

2. The Newton-Raphson Loop

Since the equilibrium equation $F_{ext} = F_{int}(u)$ is now nonlinear, we cannot solve for $u$ in a single step. We use the **Newton-Raphson method**, which linearizes the problem at each iteration $k$:

$$\mathbf{K}_t(\mathbf{u}_k) \Delta \mathbf{u} = \mathbf{F}_{ext} - \mathbf{F}_{int}(\mathbf{u}_k)$$ $$\mathbf{u}_{k+1} = \mathbf{u}_k + \Delta \mathbf{u}$$
  • $\mathbf{K}_t$ is the **Tangent Stiffness Matrix**, composed of material and geometric parts.
  • $\mathbf{R} = \mathbf{F}_{ext} - \mathbf{F}_{int}$ is the **Residual Force**, representing the out-of-equilibrium error.

3. Tangent Stiffness Components

The tangent stiffness $\mathbf{K}_t$ is split into two physical effects:

$$\mathbf{K}_t = \mathbf{K}_{mat} + \mathbf{K}_{geo}$$
  • $\mathbf{K}_{mat}$: The standard stiffness, but evaluated at the current deformed state.
  • $\mathbf{K}_{geo}$: The **Geometric Stiffness** (or stress-stiffening), which accounts for how internal stresses resist or assist further deformation (like a guitar string becoming stiffer when pulled).

4. Numerical Stability & Jacobian Locking

A common issue in Isogeometric Analysis (IGA) occurs at corners where multiple control points are coincident (degenerate mapping). While this ensures a sharp geometric corner, it creates a Singular Jacobian.

The Old Method (Singular)

Placing multiple control points at the exact same coordinate $(L, L)$ causes the area element to vanish:

$$\det(\mathbf{J}) \to 0 \implies \mathbf{K} \propto \frac{1}{\det(\mathbf{J})} \to \infty$$

Result: Stiffness Locking. The boundary nodes appear "blue" (fixed) because their stiffness is numerically infinite.

The New Solution (Nudged)

We apply a tiny Micro-Nudge ($0.01\,\text{mm}$) to the redundant points. This restores a valid mapping while maintaining visual accuracy:

$$\det(\mathbf{J}) > \epsilon \implies \mathbf{K} < \text{Tol}$$

Result: Physical Movement. The boundary is now flexible and correctly responds to traction loads.
