1D Core Engine: Mathematical Basis

B-spline Basis Functions

The core of Isogeometric Analysis (IGA) relies on B-spline basis functions, which are defined recursively using the **Cox-de Boor formula**:

$$N_{i,p}(\xi) = \frac{\xi - \xi_i}{\xi_{i+p} - \xi_i} N_{i,p-1}(\xi) + \frac{\xi_{i+p+1} - \xi}{\xi_{i+p+1} - \xi_{i+1}} N_{i+1,p-1}(\xi)$$

Technical Implementation

  • **Recursive Evaluation**: The engine implements this recurrence formula in JavaScript. For a degree $p$, it takes $O(p^2)$ operations per evaluation point.
  • **Knot Vector ($\Xi$)**: The distribution of knots determines the continuity and range of influence of each basis function. We use *Open Uniform* knot vectors to ensure interpolation at the boundaries.
  • **Real-time Rendering**: To achieve 60 FPS, the app pre-calculates the basis functions across a fine sampling grid (200+ points) and renders them to an HTML5 Canvas using direct stroke paths.

This 1D engine serves as the foundational "Online Phase" for the Digital Twin, allowing for instantaneous updates when structural parameters change.
