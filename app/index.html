Mathematical Foundation

1. B-spline Basis Functions

The foundation of IGA is the Cox-de Boor formula, which defines parametric basis functions of degree $p$ recursively:

$$N_{i,p}(\xi) = \frac{\xi - \xi_i}{\xi_{i+p} - \xi_i} N_{i,p-1}(\xi) + \frac{\xi_{i+p+1} - \xi}{\xi_{i+p+1} - \xi_{i+1}} N_{i+1,p-1}(\xi)$$

2. From B-Splines to NURBS

To exactly represent conic sections (circles, ellipses) and add local control through weights $w_i$, we use Non-Uniform Rational B-Splines (NURBS):

$$R_{i,p}(\xi) = \frac{N_{i,p}(\xi) w_i}{\sum_{j=0}^{n} N_{j,p}(\xi) w_j}$$

3. Geometry Construction

The final physical curve $\mathbf{C}(\xi)$ is the linear combination of these rational basis functions and a set of Control Points $\mathbf{P}_i$:

$$\mathbf{C}(\xi) = \sum_{i=0}^{n} R_{i,p}(\xi) \mathbf{P}_i$$

Software Logic

  • Dynamic Interaction: Dragging control points $\mathbf{P}_i$ in the UI performs a real-time re-evaluation of the curve mapping.
  • Rational Pull: Increasing $w_i$ "pulls" the curve closer to control point $\mathbf{P}_i$, a key feature of NURBS over standard B-splines.

Degree Elevation & Least Squares Fitting

1. Theoretical Goal

The goal of p-refinement (Degree Elevation) is to increase the polynomial degree $p$ of the curve to $p+1$ while keeping the exact same physical geometry $\mathbf{x}(\xi)$.

2. Least Squares Projection

This engine uses a Least Squares optimization algorithm to interpolate the new control points $\mathbf{P}_i'$ given a dense array of sampled points $S_k$ from the geometric curve:

$$\min \sum_{k=0}^{M} \left( \sum_{i=0}^{n'} R_{i, p+1}(\xi_k) \mathbf{P}_i' - S_k \right)^2$$

3. Why it matters

In Isogeometric Analysis, higher order polynomials give better differential accuracy for measuring stress matrices, but manually creating smooth splines from scratch is difficult. Elevating degree dynamically handles this automatically.

Knot Insertion (h-refinement)

1. Concept

Injecting a new parametric span $\bar{\xi}$ immediately within the Knot Vector array $\Xi$ inherently fragments the parametric basis, spawning a brand new control point computationally without mutating the core physical curve structural shape in anyway. This handles physical localized density configurations.

2. Mathematical Approach

A new span $\bar{\xi} \in [u_k, u_{k+1})$ interpolates exactly according to:

$$\mathbf{P}'_i = (1 - \alpha_i)\mathbf{P}_{i-1} + \alpha_i \mathbf{P}_i$$

where $\alpha_i$ are explicitly defined interpolation fractional parameters scaling via evaluating exact local knots elements boundaries.

3. Practical Significance

It allows mathematical engineers and digital twin structural analysts to selectively subdivide and pack denser nodes tightly around an area of high physical pressure/stress without having to blindly compute and expand polynomial computational cost complexity around standard stable uniform domains globally.

Vector Calculus (Derivative Quotients)

Because NURBS are rational quotients mapping control polynomials across weight boundaries sequentially, fully executing the Product/Quotient differential derivative rule analytically natively requires splitting evaluating mathematically bounded Basis differentials:

$$\mathbf{C}'(\xi) = \frac{\mathbf{A}'(\xi)W(\xi) - \mathbf{A}(\xi)W'(\xi)}{W(\xi)^2}$$

Where independently calculated domains limit $\mathbf{A}(\xi) = \sum N_{i,p}(\xi) w_i \mathbf{P}_i$ and purely non-physical mapping $W(\xi) = \sum N_{i,p}(\xi) w_i$.

By extracting these, Phase 1.4 accurately reconstructs absolutely exact precision boundary tangents natively needed when computing velocity, bending dynamics or internal strain matrix physics inside Digital Twin domains without estimation mapping falloff!

k-refinement (IGA Hyper-smoothness Mapping)

Standard finite element meshes only achieve universally $C^0$ continuity between mesh intersection spans natively. By selectively mixing and isolating continuous algorithmic Degree Elevation routines synchronously exactly immediately before Knot Insertion sequential limits execution maps geometrically complex $C^{p-1}$ matrix states effortlessly.

$$\mathbf{C}^{p-1}_{smoothness} = (p + 1)_{\text{elevated}} \to \mathbf{Knot_{split}}$$

Instead of dense localized fragmentation, k-Refinement computationally spawns huge structural limits mathematically matching structural integrity constraints efficiently.
